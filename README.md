## Description

Apis of application that helps manage tasks and progress of members in team.
This application is constantly updated with learning purpose and showing my experience to recruiters

## Tech Stack

- Language: TypeScript
- Framework: NestJS,
- Database: MongoDB
- ORM: Mongoose
- Queue/Cache: Redis
- API Documentation: Swagger
- AWS: S3 (update later)
- Others: NodeMailer (using Mailtrap to test send mail)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
